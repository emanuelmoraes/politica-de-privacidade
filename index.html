Política de Privacidade – Primehaus Tapetes

Última atualização: 1 de novembro de 2025

1. Coleta de informações

O aplicativo coleta apenas os dados informados voluntariamente pelo usuário ao realizar um pedido: nome, telefone e e-mail.

2. Uso das informações

Os dados são utilizados exclusivamente para contato e processamento dos pedidos, e não são compartilhados com terceiros.

3. Armazenamento e exclusão

Os dados são armazenados apenas pelo tempo necessário para o atendimento e podem ser excluídos mediante solicitação.

4. Direitos do usuário

O usuário pode solicitar acesso, correção ou exclusão dos dados pessoais, entrando em contato por: compras@primehaustapetes.com.br.

5. Segurança

Adotamos medidas de segurança para proteger os dados contra acesso não autorizado.

6. Contato

Para dúvidas, envie um e-mail para compras@primehaustapetes.com.br.
